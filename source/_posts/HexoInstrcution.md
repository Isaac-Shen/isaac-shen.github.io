## 5分钟起步

- 安装HEXO

``` bash
$ npm install hexo-cli -g
```

- 初始化项目，项目文件夹必须为空

``` bash
$ hexo init [your project folder]
```

- 运行，然后访问[http://localhost:4000](http://localhost:4000)

``` bash
$ hexo server
```
